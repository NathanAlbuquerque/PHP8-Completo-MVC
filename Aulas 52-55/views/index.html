Neste projeto das aulas 52 a 55 aprendi a usar rotas com o pacote pecee/simple-router e view templates com o pacote twig/twig.

Primeiramente preciso ter um projeto iniciado com o composer, então realizar a instalação dos pacotes informados acima.

Para as rotas, basta criar um arquivo routes.php e realizar as configurações seguindo a doc ou apenas o modelo feito aqui neste projeto.

Ja para as view templates, preciso antes criar uma classe de apoio, que carrega a pasta das view e renderiza a view, após isso configurar o metodo __construct na classe Controller e por fiz chamar esse metodo na classe filha. Pode usar a doc ou apenas seguir o mesmo modelo feito neste projeto.
